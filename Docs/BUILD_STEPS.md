# Build Steps (Summary)
- Cut 1150 mm diameter plywood base (18 mm). Drill 30 mm center pass-through.
- Print 8× outer tiles and 8× inner tiles in PETG; join with keys; screw into ribs -> base.
- Assemble 3 posts and waist ring with skate wheel rollers.
- Flash firmware and test SteamVR input mapping.
